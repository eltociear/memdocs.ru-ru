# <a name="antivirus-policy-for-endpoint-security-in-intune"></a>Антивирусная политика для обеспечения безопасности конечных точек в Intune

Антивирусные политики для обеспечения безопасности конечных точек Intune помогают администраторам системы безопасности сконцентрироваться на управлении отдельной группой параметров антивирусной программы для управляемых устройств. Чтобы использовать политику антивирусной защиты, нужно интегрировать Расширенную защиту от угроз в Microsoft Defender (ATP в Defender) с Intune в качестве решения защиты от угроз на мобильных устройствах.

Антивирусная политика включает несколько профилей. Каждый профиль содержит только те параметры, которые относятся к антивирусной программе ATP в Defender для macOS, Windows 10 или к пользовательскому интерфейсу в приложении "Безопасность Windows" на устройствах Windows 10.

Эти антивирусные политики можно найти в разделе **Управление** в узле "Безопасность конечной точки" [центра администрирования Microsoft Endpoint Manager](https://go.microsoft.com/fwlink/?linkid=2109431).

Антивирусные политики содержат те же параметры, что и профили *Endpoint Protection* или *ограничения устройств* для политики [конфигурации устройств](../configuration/device-profile-create.md), и аналогичны параметрам в политике [соответствия устройств](../protect/device-compliance-get-started.md). Однако эти типы политик включают дополнительные категории параметров, которые не связаны с антивирусной программой. Эти дополнительные параметры могут усложнить задачу настройки антивирусной программы. Кроме того, параметры в антивирусной политике, предназначенные для macOS, недоступны в других типах политик. Профиль антивирусной программы macOS заменяет необходимость настройки параметров с помощью файлов `.plist`.

## <a name="prerequisites-for-antivirus-policy"></a>Необходимые условия для антивирусной политики

- **macOS**
  - Любая поддерживаемая версия macOS
  - В Intune для управления параметрами антивирусной программы на устройстве на нем должна быть установлена ATP в Defender. См. раздел [ATP в Defender для macOS](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac) (в документации ATP в Defender)

- **Windows 10 и более поздних версий**.
  - В Intune для управления параметрами антивирусной программы на устройстве на нем должна быть установлена ATP в Defender. См. раздел [ATP в Microsoft Defender для Windows](../protect/advanced-threat-protection.md) в документации Intune.
  - Приложение "Безопасность Windows" устанавливается на всех устройствах, работающих под управлением Windows 10, и никаких дополнительных предварительных требований не существует.

## <a name="antivirus-profiles"></a>Профили антивирусной программы

**Профили macOS**.

- **Антивирус** — управление [параметрами антивирусной политики](../protect/antivirus-microsoft-defender-settings-macos.md) для macOS.

  При использовании [ATP в Microsoft Defender для Mac](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac) вы можете настраивать и развертывать параметры антивирусной программы на своих управляемых устройствах macOS с помощью Intune вместо того, чтобы настраивать эти параметры с помощью файлов `.plist`.

**Профили Windows 10**.

- **Антивирусная программа в Microsoft Defender** — управление [параметрами антивирусной политики](../protect/antivirus-microsoft-defender-settings-windows.md) для Windows 10.

  Антивирусная программа в Defender — это компонент защиты следующего поколения, который входит в состав Расширенной защиты от угроз в Microsoft Defender (ATP в Microsoft Defender). Защита следующего поколения объединяет машинное обучение, анализ больших данных, подробное исследование устойчивости к угрозам и облачную инфраструктуру для защиты устройств в вашей организации.

  Профиль *Антивирусная программа в Microsoft Defender* — это отдельный экземпляр параметров антивирусной программы, которые находятся в *профиле ограничения устройств* для политики конфигурации устройств.
  
  В отличие от параметров антивирусной программы в *профиле ограничения устройств* эти параметры можно использовать для совместно управляемых устройств. Чтобы использовать эти параметры, необходимо установить в Intune [ползунок совместного управления рабочей нагрузкой](https://docs.microsoft.com/configmgr/comanage/how-to-switch-workloads) для Endpoint Protection.

- **Интерфейс приложения "Безопасность Windows"**  — управление [параметрами приложения "Безопасность Windows"](../protect/antivirus-security-experience-windows-settings.md), которые пользователи могут просматривать в центре безопасности в Microsoft Defender, и уведомлениями, которые они будут получать. Приложение "Безопасность Windows" используется многими функциями безопасности Windows для предоставления уведомлений о работоспособности и безопасности компьютера. Уведомления приложения безопасности включают брандмауэры, антивирусные программы, SmartScreen Windows Defender и другие.

## <a name="antivirus-policy-reports"></a>Отчеты о политиках антивирусной программы

В отчетах о политиках антивирусной программы отображаются сведения об антивирусных политиках для обеспечения безопасности конечной точки и о состоянии устройств. Эти отчеты можно найти в узле безопасности конечной точки центра администрирования Microsoft Endpoint Manager.

Чтобы просмотреть отчеты, в [центре администрирования Microsoft Endpoint Manager](https://go.microsoft.com/fwlink/?linkid=2109431) перейдите в раздел "Безопасность конечной точки" и выберите **Антивирусная программа**. При выборе антивирусной программы открывается страница сводки. Дополнительные представления отчетов и состояний доступны в виде дополнительных страниц.

### <a name="summary"></a>Сводка

На странице **Сводка** вы можете [создавать новые политики](../protect/endpoint-security-policy.md#create-an-endpoint-security-policy) и просматривать список ранее созданных политик. Список содержит подробные сведения о профиле, включенном в политику (тип политики), и о том, назначена ли эта политика.

![Страница обзора антивирусной политики](./media/endpoint-security-antivirus-policy/antivirus-summary.png)

При выборе политики из списка открывается страница *Обзор* для этого экземпляра политики, на которой отображаются дополнительные сведения. При выборе плитки из этого представления Intune отображает дополнительные сведения для данного профиля, если они доступны.

![Страница обзора антивирусной политики](./media/endpoint-security-antivirus-policy/policy-overview.png)

### <a name="windows-10-unhealthy-endpoints"></a>Неработоспособные конечные точки Windows 10

На странице **Неработоспособные конечные точки Windows 10** можно просмотреть сведения о состоянии антивирусной программы для устройств Windows 10, управляемых MDM. Эта информация поступает из запущенной на устройстве антивирусной программы в Windows Defender как *состояние агента угроз*.

В этом представлении отображаются только устройства с обнаруженными проблемами. Сведения об устройствах, определенных как чистые, в этом представлении не отображаются.

![Страница обзора антивирусной политики](./media/endpoint-security-antivirus-policy/antivirus-unhealthy-endpoints.png)
